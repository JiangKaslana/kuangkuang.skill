# 志愿填报避坑检查库

Use this file for admissions-risk questions, final-check requests, and any question involving退档,调剂,提前批,野鸡大学,招生章程, or志愿提交.

## Ten Major Traps

| Trap | Risk | Required action |
|---|---|---|
| 1. 不看招生章程 | 投档成功后因身体条件、单科成绩、外语语种、性别、政审等被退档 | 每个目标学校都读当年招生章程，尤其是目标专业的特殊要求 |
| 2. 不服从调剂 | 老高考/院校+专业组模式下一次投档后被退档，可能本科掉专科 | 有调剂选项时认真判断并通常勾选；先确认省份录取模式 |
| 3. 填报过于激进 | 冲学校还冲热门专业，冲不上或被调剂到不能接受专业 | 冲稳保冷热搭配；冲只能冲自己能接受的专业组/专业范围 |
| 4. 随意填提前批 | 一旦提前批录取，后续本科批志愿作废 | 提前批是双向奔赴，不是免费抽奖；不真想去就不要填 |
| 5. 搞混大学/专业名字 | 一字之差，层次、地区、门类完全不同 | 逐字核对学校和专业名、代码、所在地、专业门类 |
| 6. 误信野鸡大学 | 学历不被认可，学费和时间损失 | 到教育部/阳光高考全国普通高校名单核验，不在名单里直接排除 |
| 7. 不看学校生活条件 | 录取后发现校区偏远、宿舍/洗浴/交通不能接受 | 查专业所在校区、宿舍、澡堂、空调、交通、校园管理 |
| 8. 用手机填志愿 | 页面显示不全、误点、漏看、密码泄露 | 尽量用电脑或大屏平板；网吧用无痕模式并清理记录 |
| 9. 保存后忘记提交 | 以为填完，实际只是草稿，截止后作废 | 提交后截图保存提交成功页和志愿确认页 |
| 10. 失去决策权 | 未来后悔时觉得人生方向不是自己选的 | 家长、老师、机构、AI 都是参考；最终确认键必须由学生自己负责 |

## Similar Name Checks

### Schools

- 哈尔滨工业大学 / 哈尔滨工程大学 / 哈尔滨理工大学
- 华东理工大学 / 东华理工大学
- 北京工业大学 / 北方工业大学
- 上海电力大学 / 上海电机学院

### Majors

- 数字媒体技术: usually计算机类/技术方向。
- 数字媒体艺术: 艺术设计方向。
- 信息与计算科学: 数学类，不是电子信息或计算机类。
- 生物科学 / 生物技术 / 生物工程 / 生物医学工程 / 生物医学科学: 名字相近但专业门类、课程和就业方向差异可能很大。

When answering these questions, insist on official专业目录,专业代码,所属门类,课程设置,毕业去向.

## 调剂 Mode Logic

| Mode | Key risk | Advice |
|---|---|---|
| 老高考 | 进档后专业满额且不服从调剂会退档 | 通常必须服从调剂；冲学校前确认调剂专业能否接受 |
| 新高考：院校+专业组 | 组内调剂到同组其他专业 | 判断专业组是否"干净"; 不干净组不要盲冲 |
| 新高考：专业+院校 | 通常无调剂 | 重点拉开梯度，不要堆在同一位次段 |

## 招生章程 Checklist

Read the current-year章程 for every target school and check:

- 专业特殊体检要求: 色盲、色弱、身高、视力、嗅觉、肝功能等。
- 单科成绩要求: 英语、数学、语文或其他。
- 外语语种限制。
- 性别限制或比例要求。
- 政审、体测、面试、口试、航海/飞行/军警类要求。
- 大类招生分流规则。
- 实验班/重点班淘汰规则、是否强制内保、能否外保。
- 转专业政策: 转出门槛和转入门槛都要看。

## Final Submission Checklist

Before clicking submit:

- Province official filling URL is correct.
- Account/password are secure; no unknown group links or QR codes.
- School names and codes are correct.
- Major names and codes are correct.
- Order is correct.
- 调剂 choice is correct where applicable.
- 保底志愿 is genuinely safe enough.
- 招生章程 special requirements are checked.
- Submitted successfully.
- Screenshots are saved.

## Answering Style

For risk questions, be direct:

- "这个坑不是小概率细节，而是会直接改变录取结果。"
- "先别讨论喜不喜欢，先确认规则上会不会被退档。"
- "能不能填提前批，关键不是多一次机会，而是你是否真的愿意被它录走。"

Then give a concrete verification path.
