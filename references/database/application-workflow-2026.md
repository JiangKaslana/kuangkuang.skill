# 2026 志愿填报实操流程

Use this file for process questions: "怎么填志愿", "需要哪些工具", "怎么定专业/学校", "位次怎么算", "冲稳保怎么安排", "有调剂/无调剂怎么填".

## 0. 准备环境

- 准备大屏浏览器，避免手机界面漏看、误点。
- 准备笔记工具，可以是纸质表格、电子表格、Notion/飞书等。
- 准备安静可思考的环境。志愿填报不是抢答题，要逐步核验。
- 2026 内容适用于大多数省份；后续年份要重新确认省份录取模式。特别注意新疆、西藏等从老高考切换到新高考的年份，以当年省考试院通知为准。

## 1. 必备信息源

| 信息源 | 用途 | 注意事项 |
|---|---|---|
| 阳光高考网 | 查院校、查专业、全国高等学校名单、各省招办入口、高考常识 | 防野鸡大学的金标准入口之一 |
| 本省教育考试院 | 一分一段表、批次线、招生政策、填报时间、征集志愿 | 不要从搜索广告、群链接、陌生二维码进入 |
| 目标院校招生网/招生公众号 | 招生章程、招生计划、往年录取、专业特殊要求 | 招生章程具有强约束力，必须读 |
| 软科 | 大学/学科/专业排名参考 | 第三方排名只能辅助，不能代替官方规则 |
| 学科评估/青塔网/研学小岛 | 中高分段看学科实力 | 第四轮官方；第五轮多为零散披露或第三方汇总 |
| 框框大学/学生经验分享 | 主观体验、专业真实感受 | 参考年份要尽量新，最好看 2025/2026 附近 |
| 大学生活质量指北 | 校区位置、宿舍、澡堂、食堂、生活条件 | 注意具体专业所在校区，不只看学校主校区 |
| 志愿工具 App | 快速筛选、查数据、做初稿 | 买多个也要交叉验证，不能让固定算法替你决策 |

## 2. 专业选择：用排除法，不用朴素兴趣直接决定

兴趣、MBTI、霍兰德等测试可以用于排除明显不适合的方向，但不能单独决定专业。高中学科兴趣和大学专业实情不是一回事。

### 三次排除

1. 用选科排除：文科/物理/物化等先决定可选专业范围。
2. 用身体、单科、政审等硬条件排除：对照体检结果和招生章程。
3. 用学科偏好和性格测试做排除：例如极度排斥数学、物理、血液、户外、倒班等。

### 定专业下限

先确定一个可接受的专业范围，再确定底线。问题不是"我最想读什么", 而是"我最低能接受到哪里".

例：想读新工科时，计算机、软件、自动化、机器人工程、电子信息可能都能接受；机械工程是否能接受，要单独问自己。能接受机械的人，含机械的专业组是"干净"的；不能接受机械的人，含机械的专业组就不干净。

## 3. 院校选择：先圈定位次区间，再看关键指标

不要试图研究全国所有学校。先用位次圈定可能够得着的学校，再深挖 20-30 所左右。

重点指标：

- 学校层次：985、211、双一流、省属重点、行业特色强校。
- 保研率：比考研率和留学率更硬，因为名额相对固定。
- 优势学科：学科评估、双一流学科、博士点。
- 硕博点：中流 211 以下尤其要看博士点，说明该学科资源、导师、平台更强。
- 重点班/实验班/大类招生：问清楚分流规则、淘汰规则、是否强制内保、能否外保。
- 转专业政策：同时看转出门槛和转入门槛。
- 城市因素：产业、实习岗位、气候、饮食、文化、交通基建、生活成本。

## 4. 录取模式判断

| 模式 | 关键点 | 策略 |
|---|---|---|
| 老高考 | 通常有院校调剂 | 服从调剂非常关键；冲学校会带来专业调剂风险 |
| 新高考：院校+专业组 | 组内调剂 | 判断专业组是否"干净"; 不干净时谨慎冲 |
| 新高考：专业+院校 | 无调剂 | 重点拉开梯度，逐个专业院校组合排序 |

## 5. 确定标杆院校的三种方法

### 方法 A：同位分/位次法

1. 查今年一分一段表，得到今年位次。
2. 到去年/前年/大前年一分一段表，找到相同位次对应分数。
3. 看该分数在往年能录到哪些学校和专业。

优点：保守稳妥。缺点：在扩招背景下可能偏保守。

### 方法 B：线差法

适合分数在某条批次线附近约 20 分以内的学生。

公式：今年分数 - 今年批次线 = 线差。用往年批次线 + 线差，对应往年可参考分数。

例：今年 500，本科线 480，线差 20；去年本科线 470，则参考去年 490。

### 方法 C：位次转换法

用于把考生数、招生计划、批次线位次变化一起纳入，结果可能更积极。

公式：

```text
今年位次 / 今年批次线位次 = 往年对应位次 / 往年批次线位次
```

例：今年 520 分对应 10000 名，特控线 500 分对应 12000 名；往年特控线对应 15000 名，则 `10000 / 12000 = X / 15000`，求得 X 后查往年一分一段表和录取结果。

如果家庭风险承受低，优先使用位次法；如果能接受一定进取，可以参考位次转换法并再做保底。

## 6. 冲稳保策略

不要机械套用 1/3 冲、1/3 稳、1/3 保。

### 有调剂模式

- 如果对专业有要求，"不冲"也是智慧。
- 推荐更多放在稳和保，例如 60% 稳、40% 保。
- 只冲干净专业组；不干净专业组冲上去，大概率面临冷门专业调剂。
- 如果只在乎学校平台、能接受冷门专业，才可以提高冲的比例。

### 无调剂模式

- 按专业+院校逐项排序。
- 每个志愿之间拉开梯度，不要连续堆同一分数段。
- 保底志愿必须低到足够稳，不能只做心理安慰。

## 7. 最后提交前检查

- 学校名称、专业名称、院校代码、专业代码逐字核对。
- 招生章程中身体条件、单科成绩、外语语种、性别、政审等要求逐项核验。
- 有调剂选项的省份，确认是否勾选服从调剂。
- 提交后截图保存：提交成功页、志愿确认页。
- 不要用陌生链接、陌生二维码、非官方网站填报。
