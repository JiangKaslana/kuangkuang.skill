### 对于学生和家长

1. **找到相似场景**：浏览上表，找到与你情况相似的示例
2. **理解决策逻辑**：看框框是如何一步步分析问题的

### 对于贡献者

## ✅ 好的对话示例特征

一个有价值的对话示例应该具备：

### 1. 清晰的背景信息
- 学生的分数段
- 省份和高考模式（老高考/新高考）
- 经济条件、家庭资源等关键因素
- 具体的困惑点

### 2. 结构化的问答
- 学生的问题明确具体
- 框框的回答有逻辑层次
- 引用了相关的心智模型或启发式规则

### 3. 实用的决策指导
- 不是简单的"选这个"，而是"如何权衡"
- 提供了思考框架和判断标准
- 强调了最终决策权在学生手中

### 4. 关键决策点分析
- 在对话后总结了核心决策逻辑
- 指出了应用的方法论
- 说明了对其他学生的参考价值

## 📝 示例文件结构

每个示例文件遵循以下结构：

```markdown
# 示例标题

## 背景
- 学生情况
- 分数/省份/模式
- 关键约束条件

## 对话记录

**学生**：[问题]

**框框**：[回答]

[完整对话]

## 关键决策点
1. 应用的心智模型
2. 关键判断逻辑
3. 风险提示

## 方法论应用
指出对话中使用了SKILL.md的哪些部分

## 参考价值
说明这个案例适合什么样的学生参考

## 延伸思考
[可选] 提供进一步的思考方向
```

## 💡 学习路径建议

### 新手入门
1. 先读 [02-avoiding-traps.md](02-avoiding-traps.md)，了解基本的避坑原则
2. 再读 [01-school-vs-major.md](01-school-vs-major.md)，理解核心权衡逻辑
3. 最后读 [05-decision-framework.md](05-decision-framework.md)，学习综合决策

### 深入学习
1. 根据自己的分数段选择对应示例（01或03）
2. 根据自己的专业倾向选择对应示例（04）
3. 回到 [SKILL.md](../SKILL.md) 系统学习完整方法论

### 实践应用
1. 用自己的情况套用示例中的分析框架
2. 向AI提出你的具体问题
3. 对比AI的回答和示例中的逻辑
4. 不断优化你的提问方式

## ⚠️ 重要提醒

1. **示例仅供参考**：不要照搬结论，每个人的情况都不同
2. **决策权在你**：框框的建议是参考，最终决策必须自己做
3. **信息会过时**：政策和专业热度会变化，注意时效性
4. **无法预测未来**：任何建议都基于当前信息，四年后不一定准确

**核心原则**：学习思考方法，而非寻找标准答案。
