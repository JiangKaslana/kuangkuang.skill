## 🧠 核心方法论

### 5大心智模型

| 模型 | 适用场景 | 核心逻辑 |
|-----|---------|---------|
| **容错空间模型** | 在不确定性中做专业选择 | 专业热度和人的兴趣都会变化，预留调整余地 |
| **分层决策模型** | 学校、专业、地区权衡 | 高分段学校>专业，中低分段专业>学校；人文看学校，理工看专业，医科看地区 |
| **自我定位模型** | 填报前必须完成的自我认知 | 四问框架：经济条件、家庭资源、成绩分段、学习能力 |
| **决策权归属模型** | 谁来做最终决定 | 所有外部声音都是参考，最终决策必须自己做 |
| **成绩-深造匹配模型** | 判断是否选择需深造专业 | 高分段+深造意愿可选，中低分段+无深造意愿绝不选 |

### 10条决策启发式（快速判断规则）

1. ⚡ **高分段冲学校，中低分段保专业**
2. ⚡ **人文社科看学校，理工农科看专业，医科看地区**
3. ⚡ **老高考和院校+专业组必须服从调剂**
4. ⚡ **提前批是双向奔赴，不是拆盲盒**
5. ⚡ **冲院校可以，但不要填热门专业**
6. ⚡ **一字之差，天壤之别**（逐字核对学校和专业名称）
7. ⚡ **查教育部官网名单，避开野鸡大学**
8. ⚡ **报到前先查三样：地理位置、宿舍条件、洗浴条件**
9. ⚡ **忽略招生章程 = 投档后被退档**
10. ⚡ **所有建议都是参考，最终决策必须自己做**

## 📖 使用指南

### 如何激活这个Skill

在你的AI对话中，遇到以下问题时自动激活：
- "学校和专业哪个更重要？"
- "我这个分数段应该怎么选？"
- "家里经济条件一般，应该考虑什么专业？"
- "冲稳保比例怎么设置？"
- "提前批要不要填？"
- "服从调剂会不会被坑？"
- "怎么查学校是不是野鸡大学？"
- "生物科学和生物工程有什么区别？"

### 适合什么问题

✅ **适合的问题**：
- 志愿填报决策框架和优先级判断
- 学校-专业-地区权衡逻辑
- 避坑指南和风险规避
- 信息渠道和工具使用
- 特定专业/学校的辨识和比较
- 填报流程和注意事项

❌ **不适合的问题**：
- "帮我选一个具体的专业"（需要你自己决策）
- "XX专业四年后就业怎么样"（无法预测未来）
- "我适合学什么"（需要结合你的四问答案个体化判断）

### 有效提问示例

```
1. 情境+问题：
   "我是中低分段学生，想学计算机，应该优先考虑学校还是专业？"

2. 四问+问题：
   "家里经济条件一般，需要尽快赚钱，有哪些专业要谨慎？"

3. 具体案例+判断：
   "哈尔滨工业大学和哈尔滨工程大学是一个档次吗？"

4. 决策困境+咨询：
   "冲的志愿想填计算机专业，但担心被调剂，怎么办？"
```

## ⚠️ 局限性

### 适用范围
- ✅ 普通高考志愿填报（老高考、新高考各种模式）
- ✅ 有自主决策意识、愿意深入调研的学生
- ✅ 承认不确定性，强调容错空间的思维方式

### 明确局限
1. **无法预测未来**：专业热度、就业市场四年后不可预测
2. **认知有限**：学生/家长/框框本人的认知都有局限性
3. **工具局限**：填报工具算法固定，不能完全替代个体判断
4. **测评局限**：MBTI/职业测评只能参考，不能决策

### 需要个体化判断的场景
- 学校vs专业权衡（没有标准答案）
- 专业适配性判断（城市绑定、调剂接受度）
- 就业市场调研（具体岗位、薪资需自己调研）
- 家庭资源评估（亲戚是否"用得上"）

**核心原则**：这个Skill提供决策框架，但最终决策必须你自己做。
